# How to Message to Approach Recruiters

When reaching out to recruiters after applying for a job, it's important to be concise, professional, and express genuine interest in the role and company. Here's a sample message you can use as a template:

`Hi [Name], I just applied for the Backend Engineer role. I’m a Software Engineering student working with FastAPI and event-driven systems, and I’d really appreciate the chance to connect and learn more about the team.`
